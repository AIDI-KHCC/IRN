Getting Started

User Registration and Profile Requirements

Required Documents

  • Good Clinical Practice (GCP) Certificate
    • Must be valid and not expired
    • System tracks expiration dates
  • Curriculum Vitae (CV)
    • Must be current and signed
    • Include research experience
  • For KHCC Investigators Only:
    • Qualitative Record Certificate (QRC)
    • Consent Training Certificate (CTC)

Submission Process

Starting a New Submission

Initial Setup

  1. Click "Start New Submission" from dashboard
  2. Enter basic information:
    • Study title
    • Study type selection
    • Primary investigator details
    • IRB number (if applicable)

Research Team Setup

  1. Add co-investigators:
    • Define roles and permissions
    • Set document access levels
    • Specify editing rights
  2. Add research assistants:
    • Set access permissions
    • Define communication rights

Form Completion

  1. Complete dynamic forms in sequence
  2. Upload required documentation
  3. Save progress (draft mode available)
  4. Validate all sections before submission

Submission Statuses and Workflow

  • Draft
    • Initial state
    • Can be edited freely
    • Not visible to reviewers
  • Submitted
    • Under initial review
    • Locked for editing
    • Awaiting reviewer assignment
  • Revision Requested
    • Changes required
    • Specific feedback provided
    • Temporarily unlocked for editing
  • Under Revision
    • Active revision in progress
    • New version being created
    • Previous version archived
  • Accepted
    • Final approval granted
    • IRB number assigned
    • Ready for implementation
  • Rejected
    • Submission not approved
    • Detailed rejection reasons provided
    • New submission may be required

Review Process

For OSAR Members

Review Assignment

  1. Access Review Dashboard
  2. View submissions needing review
  3. Create review requests:
    • Select appropriate reviewers
    • Set review deadlines
    • Specify forms for review
    • Add review instructions

Review Monitoring

  • Track review status:
    • ⏳ Pending
    • ✅ Completed
    • ❌ Declined
  • Monitor deadlines and progress
  • Handle extension requests

For Reviewers

Review Process

  1. Receive review request notification
  2. Accept or decline review assignment
  3. Access assigned submissions
  4. Complete review by deadline:
    • Provide detailed feedback
    • Submit recommendations
    • Request clarifications if needed

IRB Decision Process

Available Decisions

  • Approve
    • Grant final approval
    • Assign IRB number
    • Generate approval documentation
  • Request Revision
    • Specify required changes
    • Set revision deadline
    • Unlock for editing
  • Reject
    • Provide detailed reasoning
    • Include recommendations
    • Explain appeal process
