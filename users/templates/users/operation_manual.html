Login & Profile Management

Profile Access & Management

  • Access your profile via your name in the top right corner
  • Click "Profile" to update your information
  • Access profile using username or email

Required Documents

  • Good Clinical Practice (GCP) Certificate
  • Qualitative Record Certificate (QRC)
  • Consent Training Certificate (CTC)
  • Curriculum Vitae (CV)

All documents must be valid and up-to-date to submit applications

Document Management

  • System tracks document expiration dates
  • Indicates valid/invalid status
  • Access document history through "View Document History"

Dashboard & Submissions

My Submissions Dashboard

View all submissions where you are involved as:

  • Primary Investigator
  • Co-Investigator
  • Research Assistant

Submission Information Display

  • System-generated ID
  • KHCC number (assigned after OSAR review)
  • Study title
  • Primary investigator name
  • Version number (updates with each submission)
  • Creation date
  • Last modification date

Available Actions

  • Edit submission (when permissions allow)
  • View version history
  • Download PDF versions
  • Compare different versions
  • View actions history
  • Review decisions
  • Additional actions:
    • Withdraw study
    • Submit progress report
    • Request study amendment
    • Process study closure

Submission Process

Starting a New Submission

  1. Provide study title
  2. Select study type (determines required forms)
  3. Specify primary investigator role
  4. Add team members:
    • Research assistants (specify permissions for editing, submission, communication)
    • Co-investigators (define roles and access)
  5. Complete required forms based on study type
  6. Upload supporting documents (unlimited)
  7. Review submission quality using KHCC Brain
  8. Submit application

Review Process

Reviewer Access

  • OSAR members can:
    • View group submissions
    • Request reviews from IRB/Research Council
    • Assign KHCC numbers
  • IRB/Research Council members can:
    • View assigned submissions
    • Leave notes (visible to other members)
    • Make decisions:
      • Request revision
      • Reject submission
      • Accept submission

Review Management

  • Track pending reviews
  • Submit reviews
  • Request deadline extensions
  • Request additional details
  • Access submission PDFs
  • View all versions and previous reviews

Messaging System Features

  • Receive system notifications
  • Email notifications for all communications
  • Complete documentation of all research-related communications
