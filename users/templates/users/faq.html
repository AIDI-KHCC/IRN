Frequently Asked Questions

Profile Access
  • Access your profile via your name in the top right corner
  • Click "Profile" to update your information
  • Access profile using username or email
Required Documents
  • Good Clinical Practice (GCP) Certificate
  • Qualitative Record Certificate (QRC)
  • Consent Training Certificate (CTC)
  • Curriculum Vitae (CV)
All documents must be valid and up-to-date to submit applications
Document Management
  • System tracks document expiration dates
  • Indicates valid/invalid status
  • Access document history through "View Document History"

My Submissions Dashboard

View all submissions where you are involved as:

  • Primary Investigator
  • Co-Investigator
  • Research Assistant
Submission Information Display
  • System-generated ID
  • KHCC number (assigned after OSAR review)
  • Study title
  • Primary investigator name
  • Version number (updates with each submission)
  • Creation and modification dates
Available Actions
  • Edit submission (if permissions allow)
  • View version history and pending documents
  • Compare different versions
  • View actions history
  • Generate PDF
  • Archive submission
Submission Process
  1. Enter title and select study type
  2. Specify roles (PI, co-investigator, research assistant)
  3. Set permissions for research assistants
  4. Complete required forms based on study type
  5. Review qualifications and missing fields
  6. Upload supporting documents
  7. Optional quality check using KHCC Brain
  8. Submit application

Reviewer Access
  • OSAR, IRB, and Research Council members see group submissions
  • OSAR can request reviews from IRB/Research Council heads
  • Reviews can be forwarded to privileged members
Review Features
  • KHCC number assignment
  • Note system with visual indicators
  • Review submission options:
    • Request revision
    • Accept submission
    • Reject submission
  • Manage pending reviews
  • Request extensions or additional details
  • View submission versions and previous reviews

Messaging System Features
  • Standard inbox functionality
  • Compose new messages
  • View sent messages
  • Receive system notifications
  • Email notifications for all communications
All research-related communications must use the system messaging for documentation purposes.
Additional Resources
  • Operation manual for detailed guidance
  • FAQ section for common questions
  • Feedback system for suggestions and issues
