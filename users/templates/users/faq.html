Frequently Asked Questions

Your submission will be reviewed by the research office on the next working day after submission. The system automatically notifies OSAR members when new submissions are received.

Initial review typically begins within 24 hours (on working days).

The review timeline depends on the type of submission but should be completed within a maximum of 4 weeks.

  • Initial review: 1-2 working days
  • Full review process: Up to 4 weeks
  • Revision reviews: 1-2 weeks
Important: When you receive a revision request, please respond as soon as possible to maintain timeline efficiency.

Security Measures
  • Data Encryption
    • All data is encrypted at rest using AES-256 encryption
    • SSL/TLS encryption for all data in transit
    • Secure file storage with encrypted backups
  • Access Control
    • Role-based access control (RBAC)
    • Multi-factor authentication (MFA)
    • IP-based access restrictions
    • Automatic session timeout
  • Audit Trail
    • Complete activity logging
    • Version control for all documents
    • User action tracking
  • Compliance
    • HIPAA compliance measures
    • Regular security audits
    • Automated vulnerability scanning
The system is hosted on secure KHCC servers with enterprise-level security protocols.
