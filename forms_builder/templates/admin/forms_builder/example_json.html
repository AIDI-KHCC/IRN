JSON Format Guide

Each field in your form should include the following properties:

  • name: The label/name of the field
  • field_type: One of: text, number, date, choice, dropdown
  • default_value: Initial value (can be empty)
  • max_length: Maximum length for text fields (null for other types)
  • choices: Array of options for choice/dropdown fields
  • required: true/false - whether the field is mandatory
  • help_text: Explanatory text shown below the field
